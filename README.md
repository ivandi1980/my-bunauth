# Next Lucia (bun)

## Description

Build the Authentication system in React with Typescript, using Lucia-Auth

Run Command to Install Next :

```bash
bunx create-next-app@latest
```

Run The App :

```bash
bun dev
```

Install Shadcn UI :

```bash
bunx shadcn-ui@latest init
```
